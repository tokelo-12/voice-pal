# **App Name**: VoicePal

## Core Features:

- Voice Command Interface: A minimalist 'Blob' UI that acts as a visual and interactive anchor for all voice commands.
- Sesotho/isiZulu Command Translation Tool: Leverages Google Translate API to interpret spoken Sesotho and isiZulu voice commands into actionable system requests.
- Call Management: Initiate phone calls to contacts (either saved or new numbers) through clear voice commands, including call termination.
- SMS Composition & Sending: Dictate, review, and send SMS messages to specified contacts or numbers, with voice-guided correction options.
- Airtime Purchase: Purchase airtime by specifying amount and recipient (self or other), confirmed through voice.
- Audio Feedback System: Provides audible confirmations, instructions, and error messages to guide the user without visual reliance.

## Style Guidelines:

- Color scheme: Dark theme with high contrast for enhanced accessibility for visually impaired users.
- Primary color (actions, key elements): A balanced and trusting blue (#2E90E6).
- Background color: A deep, muted dark blue-gray (#15191A) to reduce eye strain.
- Accent color (active states, alerts): A bright and vivid cyan (#BFFCFC) for clear indication and contrast.
- Body and headline font: 'Inter' (sans-serif) for optimal legibility and a modern, neutral feel across all textual outputs.
- Focus on highly minimalist, abstract shapes for the 'Blob' and state indicators, prioritizing clarity and immediate recognition through color and subtle motion.
- Extremely minimal screen layout with a singular, central floating 'Blob' element; voice commands manage all navigational and functional aspects.
- Subtle, fluid animations for the 'Blob' to indicate microphone status (listening, processing, speaking) and feedback (success, error), serving as primary non-verbal communication cues.
